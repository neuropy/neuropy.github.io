neuropy

A command-line based Python program for interactive analysis of sorted neuronal spike data, stimulus data, and LFP waveforms.

The latest version is available in its GitHub repository.

Here's a screenshot showing the spatiotemporal spike-triggered average (STA) response of each sorted cell (right, rows) to an msequence noise display, as well as the spectrogram of the associated local field potential (LFP).
